# SCRMTracker
## Introduce
StarCraft: Remastered Player Tracker is open-sourced SC:RM player management software powered by NPCAP library.

## Feature
* Display joined player's BattleTag, SC BattleNet username, Masked IPv4 address, comment, blacklis
* Automatically add player information to SQLite database
* Manage player information like BattleTag, username, SHA-256 hashed IP address, comment, blacklist boolean
* Track blacklisted player by comparing battletag, IP address hash

## Required before use
* [.NET 8.0 Runtime](https://builds.dotnet.microsoft.com/dotnet/Runtime/8.0.22/dotnet-runtime-8.0.22-win-x64.zip)
* [NPCAP NDIS Driver](https://npcap.com/dist/npcap-1.85.exe)
